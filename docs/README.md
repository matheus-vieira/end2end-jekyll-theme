# Project Documentation

This folder contains focused documentation pages for subsystems of the
`end2end-jekyll-theme` project (comments, Rake tasks, etc.). Use the links
below to find configuration details and setup steps.

- `comments/` — guidance for comments providers (`giscus`, `disqus`) and how
  to configure them (per-site and per-page).
- `rake/` — documentation for `Rakefile` tasks used for building and serving
  the site locally.

See the individual folders for details.
